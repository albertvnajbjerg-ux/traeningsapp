# Træn

En lille offline træningsapp til iPhone. Appen er lavet som en statisk PWA, så den kan hostes gratis på GitHub Pages og lægges på hjemmeskærmen via Safari.

## Brug på iPhone

1. Åbn GitHub Pages-linket i Safari.
2. Tryk på Del-knappen.
3. Vælg "Føj til hjemmeskærm".
4. Åbn appen fra hjemmeskærmen.

Appen gemmer profil, plan, noter og status lokalt på telefonen.
